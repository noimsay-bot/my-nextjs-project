Codex 작업 시작 지시:

이 폴더의 `schedule-integrated-v10.html`을 근무표 기능의 기준 파일로 삼아 React/Next.js 구조로 리팩토링해줘.
사용자가 제공한 순번표는 seed 기본값으로 유지해줘.
근무표 기능이 가장 중요하므로 먼저 이것부터 안정화하고,
그 다음 영상평가 제출 / 평가페이지 / 팀장페이지 / 관리자페이지를 붙여줘.

중요:
- 기존 근무표 기능 삭제 금지
- 연장은 주 단위
- 평일/주말/휴일 분기 정확히 유지
- 오프 기능, 자동 재배치, 스냅샷, 수정 모드, 드래그 이동 유지
- 파일별 변경사항과 실행 방법을 같이 설명해줘
